# Практические задания

## Уровень 1: Базовые команды

### Задание 1: Начало работы
1. Создайте новую папку `git-practice`
2. Инициализируйте git репозиторий
3. Проверьте наличие папки .git

### Задание 2: Работа с файлами
1. Создайте файл `hello.txt` с текстом "Hello Git!"
2. Добавьте файл в staging area
3. Сделайте коммит с сообщением "Add greeting file"

## Уровень 2: Ветвление

### Задание 3: Создание веток
1. Создайте ветку `feature/add-menu`
2. Добавьте файл `menu.txt`
3. Переключитесь обратно в main
4. Объедините изменения

## Уровень 3: Работа с историей

### Задание 4: Анализ истории
1. Сделайте 3 разных коммита
2. Просмотрите историю в разных форматах
3. Найдите конкретный коммит по сообщению
